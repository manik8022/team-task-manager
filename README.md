# Team Task Manager

## Description
A full-stack task management web application with role-based authentication for Admin and Members.

## Features
- Admin & Member Login
- JWT Authentication
- Task Creation
- Task Assignment
- Task Status Tracking
- Overdue Tracking
- Dashboard Analytics
- Light/Dark Mode
- Responsive UI

## Tech Stack

### Frontend
- React.js
- CSS

### Backend
- Flask
- Flask JWT
- SQLAlchemy

### Database
- SQLite

## Deployment
- Frontend: Vercel
- Backend: Render
